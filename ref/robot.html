Robot

The Robot class is the main class for controlling and fetching information about your kit.

class sr.robot3.Robot(*, auto_start=True, verbose=False, env=<j5.backends.environment.Environment object>, ignored_ruggeduinos=None)[source]

Student Robotics robot.

Inherits from j5 BaseRobot to ensure that all boards are made safe in the event of a crash, and to ensure that there is only one instance of this class on a given machine.

__init__(*, auto_start=True, verbose=False, env=<j5.backends.environment.Environment object>, ignored_ruggeduinos=None)[source]

Initialise a Robot object.

Parameters

  • auto_start (bool) – Automatically start the robot ignoring the start button, defaults to False.

  • verbose (bool) – Turn on verbose logging, defaults to False.

  • env (Environment) – The j5.backends.environment.Environment to run the Robot under. See Environments for more information.

  • ignored_ruggeduinos (Optional[List[str]]) – List of Ruggeduino serial numbers to ignore. See Custom Ruggeduino Firmware for more information.
